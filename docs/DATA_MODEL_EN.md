
# Data Model – Table `BD`

This document describes the **official data model** for table/sheet `BD` used by **Exam‑Web**.

It applies to:
- Microsoft Excel
- Microsoft Access
- SQLite

## Structure (BD)

Field groups:
- Identification (`UN`, `NOMBRE`, `TIPO`)
- Content (`PREGUNTA`, `A–D`, `RA–RD`, `R`)
- State (`VIS`, `COR`, `REV`, `Filtro`)
- Statistics (`OK`, `KO`)
- Study notes (`ESTUDIO`)

Each row represents a unique question.
