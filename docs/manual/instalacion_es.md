
# Instalación

## Ejecutable (recomendado)
1. Descargue Exam‑Web
2. Descomprima el contenido en una carpeta (por ejemplo `C:\exam-web`)
3. Ejecute `Exam-Web.exe`

## Desde código fuente
```bash
pip install -r requirements.txt
python main.py
```

## Requisitos
- Windows 10 o superior
- Python 3.11+ (solo desarrollo)
