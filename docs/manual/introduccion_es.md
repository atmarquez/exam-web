
# Introducción

Exam‑Web es una aplicación de escritorio para Windows que permite ejecutar un sistema de exámenes web de forma local.
Combina una aplicación gráfica con un servidor web embebido para ofrecer entrenamiento y práctica de exámenes.

Está orientada a:
- Estudio autodidacta
- Preparación de certificaciones
- Simulación de exámenes tipo test
