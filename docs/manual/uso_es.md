
# Uso del programa

## Iniciar el servidor
- Seleccione el fichero de datos
- Pulse “Iniciar servidor”
- Acceda desde el navegador web

## Detener el servidor
- Pulse “Parar servidor”
- El puerto se libera automáticamente

## Bandeja del sistema
- Minimizar la ventana
- Restaurar la aplicación
- Salir completamente
