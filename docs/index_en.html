What is Exam‑Web?

Exam‑Web is a Windows desktop application that allows you to manage and launch a local web‑based exam server in a simple, flexible and controlled way.

The project is designed for training and simulation of multiple‑choice exams, especially in technical studies, certifications and competitive exams.

Exam‑Web clearly separates two areas: the server management (Windows application) and the exam experience (web interface).

Project architecture

  • Windows desktop application (configuration and control)
  • Local web server based on Python + Flask
  • Browser‑based exam interface
  • Question databases supported in:
    • Excel
    • Access
    • SQLite

Windows application (Control panel)

The Windows application is the main entry point of the system. From it, the entire web server lifecycle is controlled.

  • Select the question data file
  • Automatic detection of the data source type
  • Configure port and protocol (HTTP / HTTPS)
  • Configure TLS certificates for HTTPS
  • Start and stop the web server
  • Minimize to system tray
  • Access real‑time log console
  • Persist configuration in a JSON file

The application can run in the background, allowing users to access the exam server from the browser at any time.

Web exam interface

Once the server is running, users access the exam through a standard web browser.

  • Multiple‑choice question display
  • Randomized answer content
  • Per‑question timer with progressive visual alerts
  • Tracking of correct and incorrect answers
  • Per‑question performance history
  • Marked questions for review
  • Access to glossary, figures and annotations

The web interface is designed to be clear and focused exclusively on the exam experience.
