¿Qué es Exam‑Web?

Exam‑Web es una aplicación de escritorio para Windows que permite gestionar y lanzar un servidor web de exámenes de forma sencilla, flexible y local.

El proyecto está pensado para la práctica y simulación de exámenes tipo test, especialmente en contextos de estudio técnico, oposiciones o certificaciones.

Exam‑Web separa claramente dos ámbitos: la gestión del servidor (aplicación Windows) y la realización del examen (interfaz web).

Arquitectura general del proyecto

  • Aplicación de escritorio Windows (control y configuración)
  • Servidor web local basado en Python + Flask
  • Interfaz web accesible desde el navegador
  • Soporte de bases de datos de preguntas en:
    • Excel
    • Access
    • SQLite

Aplicación Windows (Panel de control)

La aplicación Windows es el punto de entrada principal del sistema. Desde ella se controla completamente el servidor web.

  • Selección del fichero de datos de preguntas
  • Detección automática del tipo de base de datos
  • Configuración de puerto y protocolo (HTTP / HTTPS)
  • Configuración de certificados para HTTPS
  • Inicio y parada del servidor web
  • Minimización a la bandeja del sistema
  • Acceso a consola de logs en tiempo real
  • Persistencia de configuración en fichero JSON

La aplicación puede funcionar en segundo plano, permitiendo acceder al servidor desde el navegador en cualquier momento.

Interfaz web del examen

Una vez iniciado el servidor, los usuarios acceden al examen a través del navegador web.

  • Visualización de preguntas tipo test
  • Aleatorización del contenido de las respuestas
  • Temporizador por pregunta con alertas visuales
  • Seguimiento de aciertos y fallos
  • Registro histórico por pregunta
  • Marcado de preguntas para revisión
  • Acceso a glosario, gráficos y anotaciones

La interfaz web está diseñada para ser clara, usable y enfocada exclusivamente en la experiencia del examen.
