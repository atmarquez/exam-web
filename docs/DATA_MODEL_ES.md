
# Modelo de Datos – Tabla `BD`

Este documento describe el **modelo de datos oficial** de la tabla/hoja `BD` utilizada por **Exam‑Web**.

Es válido para:
- Microsoft Excel
- Microsoft Access
- SQLite

## Estructura (BD)

Campos definidos:
- Identificación (`UN`, `NOMBRE`, `TIPO`)
- Contenido (`PREGUNTA`, `A–D`, `RA–RD`, `R`)
- Estado (`VIS`, `COR`, `REV`, `Filtro`)
- Estadísticas (`OK`, `KO`)
- Estudio (`ESTUDIO`)

Cada fila representa una pregunta única.
